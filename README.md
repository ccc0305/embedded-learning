# 🔧 Embedded Learning - 嵌入式系统学习笔记与实践

🎯 **目标**：系统学习 C 语言、STM32、驱动开发与嵌入式系统设计，掌握从底层原理到项目开发的完整能力。

## 🧭 学习路线图（Roadmap）
-
- ✅ C语言基础（语法、数组、指针、结构体）
- ✅ 编译与链接过程（预处理 → 编译 → 汇编 → 链接）
- ✅ 单片机原理与 STM32 架构
- 🔄 外设驱动开发（GPIO、USART、ADC、PWM、I2C、SPI）
- 🔄 RTOS 入门与多任务管理
- 🔄 项目实践（串口调试助手、LED 跑马灯、定时器计数等）
